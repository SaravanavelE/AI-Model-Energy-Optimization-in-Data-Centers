# AI Model Energy Optimization in Data Centers

## Overview

This project demonstrates an **AI-driven resource management system for
modern data centers**. It monitors workloads, predicts demand using
machine learning, and dynamically allocates compute resources to reduce
**energy consumption and carbon footprint**.

## Features

-   Real-time CPU & GPU monitoring
-   Machine learning workload prediction
-   Dynamic resource scheduler
-   Energy-efficient AI model selection
-   Interactive dashboard

------------------------------------------------------------------------

## Tech Stack

Backend - Python - Flask - NumPy - Scikit-learn

Frontend - HTML - CSS - JavaScript - Chart.js

------------------------------------------------------------------------

## Run the Project

Install dependencies

pip install -r requirements.txt

Run backend

python app.py

Open dashboard

index.html
